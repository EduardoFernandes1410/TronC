# TronC
Tron game in C

### Pre-requisitos
- Esteja em um ambiente Linux\
- Rode o comando no terminal:\
```
sudo apt-get install libncurses5-dev
```

### Rodar
- Rode o comando no terminal, na pasta do jogo:\
```
make -s
```
\
